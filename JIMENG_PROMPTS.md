## 使用技巧

### 1. 提示词结构
**基础公式**：主体 + 动作 + 风格 + 质量 + 时长

**示例**：
```
主体：Mahjong Heaven游戏Logo
动作：金色立体文字从中心浮空浮现
风格：东方禅意美学，翡翠绿配色
质量：4K超高清，电影级布光
时长：3秒慢动作
```

### 2. 关键词推荐

**风格类**：
- 东方禅意美学
- 中国风水墨风格
- Premium luxury feel
- 电影级广告制作
- 优雅高贵

**颜色类**：
- 翡翠绿和金色
- 深绿色渐变
- 金色光芒
- 黑金配色

**特效类**：
- 粒子特效
- 光晕效果
- 烟花礼花
- 碎片飞溅

**质量类**：
- 4K超高清
- 电影级画质
- 广告级品质
- 专业制作水准

### 3. 参数设置

**视频比例**：
- 16:9（横屏，YouTube、App Store）
- 9:16（竖屏，TikTok、Instagram Reels）

**运动幅度**：
- 3-5（缓慢、优雅）
- 6-8（中等、流畅）
- 9-10（快速、动感）

**视频时长**：
- 3秒（标准）
- 5秒（最长）

### 4. 批量生成

**推荐流程**：
```
1. 生成5-10个不同片段
2. 选择质量最好的
3. 下载保存
4. 用剪辑软件拼接
5. 添加转场和音乐
6. 导出完整视频
```
